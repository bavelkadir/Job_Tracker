# 🧾 Job Tracker – C# Console Application

## 📌 Projektbeskrivning
Detta projekt är en C# Console-applikation som används för att hantera och följa upp jobbansökningar.  
Programmet låter användaren lägga till, visa, filtrera, sortera, uppdatera och ta bort jobbansökningar via ett menybaserat gränssnitt i konsolen.

Varje jobbansökan innehåller:
- Företagsnamn
- Tjänst / position
- Status (Applied, Interview, Offer, Rejected)
- Ansökningsdatum
- Svarsdatum (om svar har mottagits)
- Löneförväntan

Projektet använder **LINQ** för filtrering, sortering och statistik samt ett användarvänligt gränssnitt med piltangenter, färger och emojis.

---

## ▶️ Hur man kör programmet
1. Klona repot från GitHub
2. Öppna projektet i **Visual Studio** eller **Visual Studio Code**
3. Kontrollera att projektet är ett **Console App (.NET)**
4. Kör programmet genom att:
   - Klicka på **Run / Start**
   - eller köra:
     ```bash
     dotnet run
     ```
5. Navigera i menyn med **↑ / ↓** och välj alternativ med **Enter**
